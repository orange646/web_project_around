# Proyecto 7: Alrededor de los EE. UU.

Este proyecto es una página web interactiva desarrollada como parte del Sprint 7 de TripleTen.

## Descripción

La página presenta un perfil de usuario y una galería de lugares. El proyecto permite:

- editar la información del perfil
- abrir y cerrar popups
- dar like a las tarjetas
- eliminar tarjetas
- crear nuevas tarjetas

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Metodología BEM
- Diseño responsive
- Git y GitHub
